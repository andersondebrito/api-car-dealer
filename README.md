# api-car-dealer

# Technologies
## Back end
- Java
- Spring Boot
- JPA / Hibernate
- Maven
- H2 database
- Junit
- Mockito

# How to run the project

## Back end
Prerequisites: Java 11

```bash
# clone repository
git clone https://github.com/andersondebrito/api-car-dealer.git

# enter the project folder
cd api-car-dealer-main

# run the project
./mvnw spring-boot:run
```
